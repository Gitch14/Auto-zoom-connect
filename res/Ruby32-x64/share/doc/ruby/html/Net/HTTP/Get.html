    class Net::HTTP::Get

    Class for representing HTTP method GET:

    require 'net/http'
uri = URI('http://example.com')
hostname = uri.hostname # => "example.com"
req = Net::HTTP::Get.new(uri) # => #<Net::HTTP::Get GET>
res = Net::HTTP.start(hostname) do |http|
  http.request(req)
end

    Properties:
